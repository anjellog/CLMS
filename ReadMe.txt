This is a Computer Laboratory Management System created with PHP and designed with Bootstrap.
This System has Inventory, Tracking, Security, and Report.
It uses MD5 for Encrypting the password.
Reports can be generated as PDF or directly print it.

Installation:
1. Paste the project folder inside htdocs
2. Create new database named clms
3. Import the sql found in the root directory.
4. If successful, run the localhost.


Running:
1. Run the Xampp server.
2. Start the Apache and SQL
3. On browser, type localhost/root_folder
4. That's it.
